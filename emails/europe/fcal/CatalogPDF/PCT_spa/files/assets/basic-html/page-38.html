El comprobador de peso muerto

es un método demostrado de

calibración de presión, que se

elige generalmente para las

aplicaciones de banco cuando

los requisitos principales son

la precisión y la confiabilidad.

Las calibraciones se realizan

en el banco por conveniencia y

para mantener las condiciones

de referencia. El banco es un

lugar conveniente para limpiar,

inspeccionar, calibrar y reparar

con todo el equipo disponible

que se requiera. Las condiciones

de referencia son necesarias

para lograr la precisión de

referencia del dispositivo bajo

prueba y los estándares de

calibración. Es probable que

se requiera la precisión de

referencia para mantener las

relaciones de incertidumbre de

las pruebas (TUR, por sus siglas

en inglés).
