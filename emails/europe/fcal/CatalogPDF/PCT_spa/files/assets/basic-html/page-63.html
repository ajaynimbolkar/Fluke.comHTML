CONSEJOS

TÉCNICOS

Para realizar la prueba:

Aísle el sensor del proceso.

Sumerja el sensor en una fuente de temperatura de precisión como un

pozo seco o un baño capaz de cubrir el rango de temperatura necesario.

Conecte el estándar de temperatura y la salida de 4-20 mA del transmisor

a un medidor o calibrador adecuado (por ejemplo, el sistema electrónico

de proceso de un Pozo metrológico de campo de Fluke o las entradas de

un Fluke 754).

Alimente el lazo. (El Fluke 754 y el sistema electrónico de proceso de un

Pozo metrológico de campo cuentan con esta capacidad.)

Ajuste la temperatura del baño o el pozo seco a cada uno de los puntos

de prueba. (Con Pozos de metrología de campo estos puntos de prueba se

pueden pre-programar y automatizar.)

En cada punto de prueba, controle y registre las lecturas del estándar

de temperatura y las lecturas locales o remotas conectadas a la salida

del transmisor.

Además, registre la salida de 4-20 mA del transmisor para determinar qué

dispositivo necesita ser ajustado, en caso de necesitar realizar un ajuste.
