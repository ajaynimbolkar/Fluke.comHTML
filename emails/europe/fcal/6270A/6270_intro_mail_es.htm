  Fluke Calibration presenta:
Calibrador/controlador de presión modular 6270A

Combine, seleccione y calibre.

Ahora puede calibrar una amplia variedad de dispositivos de presión con un único instrumento.

El Calibrador de presión 6270A admite la combinación de hasta cinco módulos de medición, de esta forma puede calibrar instrumentos con rangos desde el vacío a 20 MPa (200 bar). Los dos niveles de incertidumbre que se ofrecen permiten adaptar cada módulo a las necesidades y al presupuesto disponible.

El 6270A permite cambiar los módulos sin mover la unidad principal, de este modo, resulta sencillo cambiar los rangos de presión. El mantenimiento y la reparación es una operación rápida y fácil, haciendo que el tiempo de inactividad sea una preocupación del pasado.

Características del modelo 6270A de un vistazo:

  • Calibre una amplia variedad de manómetros y sensores con un único instrumento
  • La configuración modular hace de ésta una solución versátil y económica
  • Fácil de operar
  • De fácil mantenimiento
  • Amplio intervalo de medición: vacío a 20 MPa (3000 psi)
  • Dos niveles de exactitud: 0,02 % FS o 0,01 % de lectura, le permiten equilibrar entre exactitud y presupuesto
  • Velocidad alta, control de presión estable
  • Interfaz del usuario gráfica localizada en nueve idiomas a elección
  • Se puede automatizar completamente con COMPAS ® para Software de presión
  • El sistema de prevención de contaminación opcional ayuda a mantener las válvulas limpias y libres de desechos
