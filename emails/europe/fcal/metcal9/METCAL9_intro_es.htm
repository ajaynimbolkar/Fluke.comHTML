Presentación del Software de gestión de calibraciones MET/CAL, versión 9

Grandes beneficios de productividad con una nueva interfaz intuitiva
Cuando la productividad y el rendimiento importan, el software de calibración de Fluke cumple al incrementar la productividad hasta un 80 por ciento, o incluso más».

La versión 9 de MET/CAL ofrece una nueva interfaz intuitiva que brinda mayor y mejor visibilidad al proceso de calibración. De modo que los técnicos menos experimentados pueden volverse productivos rápidamente.

El software de calibración de Fluke incluye dos aplicaciones: El software MET/CAL, el líder de la industria para calibración automatizada; y el software MET/TEAM para la gestión tanto del flujo de trabajo de calibración como de los activos. Juntos conforman el sistema de automatización de calibraciones más completo, diseñado para ayudarle a lograr productividad, eficiencia y consistencia. Este sistema también le ayuda a cumplir con los requisitos de diferentes normas de calidad, como ISO 9000, ISO/IEC 17025, ANSI Z540.3, entre otras.
